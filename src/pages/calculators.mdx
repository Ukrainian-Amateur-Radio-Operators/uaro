#  Калькулятор антени Flower Pot

:::info[Інформація]

Цей калькулятор використовується для визначення довжини кожного елемента антени для Flower Pot.

Сюди входить верхній радіатор, нижній радіатор, а також довжина кабелю, який буде намотаний на ПВХ-трубу.

Тут ми не рахуємо кількість витків, тому що вона буде варіюватися в залежності від діаметра ПВХ труби, на яку намотується кабель.

Ми розраховуємо довжину кабелю, який потрібно намотати, тоді ми отримаємо кількість витків після того, як кабель буде намотаний на ПВХ-трубу.
:::

<Calculator />

:::warning[Примітки:]
- Резонансний коефіцієнт - це значення між 5-10%, значення за замовчуванням - 7,5% або середнє значення.
- Коефіцієнт укорочення кабелю є постійною величиною для кожного типу кабелю. Для кабелю RG58 постійне значення становить 0,66.
:::
